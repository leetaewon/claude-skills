# mystyle — 채널별 작법 Skill

매체별 일반적인 작법 원칙에 맞게 글을 작성한다.
이 스킬은 채널 관례(분량, 구조, 격식 수준)만 담은 범용 가이드다. 특정한 어투나 말버릇을 강제하지 않는다.

> 나만의 고유한 문체까지 반영하고 싶다면 **style-creator** 스킬로 인터뷰 기반 맞춤 버전을 만들어 이 스킬을 교체할 수 있다.

## 작동 방식

1. 사용자가 **매체를 명시**하면 → 해당 references 파일을 읽고 적용
2. 매체가 **불분명**하면 → 아래 매체 목록을 안내하고 선택을 요청
3. 글감(주제/소재)이 없으면 → 요청 후 작성
4. **카드뉴스**의 경우, 유형(정보 전달형 / 하이라이트 요약형)이 불분명하면 → 먼저 확인

| 책, 원고, 책 집필 | `references/book.md` |

## 매체별 파일 로드 규칙

매체가 결정되면 **해당 파일 하나만** 읽는다. 여러 파일을 동시에 읽지 않는다.

## 공통 원칙 (모든 매체 공통)

- 정보 나열보다 독자가 처한 상황이나 맥락을 먼저 짚어주고 시작
- "총정리", "완벽 가이드", "A to Z" 등 과장된 제목 지양
- 검증 안 된 내용은 단정적으로 쓰지 않기
- 어투(반말/존댓말, 평어체/대화체)는 강제하지 않음 — 매체 특성상 일반적인 경향만 안내
