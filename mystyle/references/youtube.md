# 유튜브 스크립트 작법 가이드

## 채널 특성

- 음성으로 전달되는 매체이므로 구어체에 가깝게 쓰는 것이 자연스럽다
- 시청자가 이탈하기 쉬운 도입부 5~10초가 특히 중요
- 화면 시연이 함께 있는 경우, 말과 화면이 같이 진행되는 흐름을 고려해야 함

## 분량 및 구조

- 영상 길이에 맞춰 분량 조정 (1분당 약 300~350자 기준)
- 오프닝은 인사보다 바로 본론으로 들어가는 편이 이탈률을 낮추는 데 유리
- 구독·좋아요 유도는 오프닝보다 중반 이후가 자연스러움

## 일반적인 스크립트 구조

```
오프닝 — 오늘 다룰 내용, 또는 시청자가 겪는 상황 제시
→ 본론 — 설명 + 화면 시연 병행
→ 정리 — 핵심 요약
→ 클로징 — 다음 영상 예고 또는 행동 유도
```

## 주의사항

- "안녕하세요 여러분, 오늘은 ~에 대해 알아보겠습니다" 같은 정형화된 인사는 이탈률을 높일 수 있음
- 글로 쓴 문장을 그대로 읽으면 어색하게 들릴 수 있어, 말하듯 자연스러운 흐름으로 다듬는 것이 중요
- 결론을 먼저 말하고 과정을 설명하는 구조가 시청 지속에 유리한 경우가 많음
