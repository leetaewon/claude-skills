# 네이버 블로그 작법 가이드

## 채널 특성

- 검색 유입이 많은 매체. 제목과 도입부에 핵심 키워드가 들어가는 것이 유리
- 정보성 콘텐츠와 일상형 콘텐츠가 함께 소비되는 플랫폼
- 사진·이미지와 함께 읽히는 경우가 많아 텍스트만으로 길게 늘어지면 가독성이 떨어짐

## 분량 및 구조

- 분량: 1,000~2,500자 권장 (정보성 글은 다소 길어도 무방)
- 소제목 2~5개로 구획을 나누는 것이 가독성에 유리
- 단락은 3~5줄 이내로 짧게

## 일반적인 글 구조

```
도입 — 검색해서 들어온 독자가 원하는 것을 짚어주기
→ 본론 — 소제목별로 정보 또는 경험 전개
→ 마무리 — 핵심 요약 또는 다음 행동 제안
```

## 제목 작성 시 고려사항

- 검색 키워드를 포함하되 너무 광고성으로 보이지 않게
- 숫자나 구체적 결과를 포함하면 클릭률에 유리한 경우가 많음

## 주의사항

- 과도한 협찬·광고성 어조는 신뢰도를 낮출 수 있음
- 정보성 글의 경우 출처나 근거를 명확히 밝히는 것이 좋음
- 사진 배치를 고려해 문단을 너무 길게 쓰지 않기
