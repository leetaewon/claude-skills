# 링크드인 작법 가이드

## 채널 특성

- 직업적 맥락에서 읽히는 매체. 페이스북보다는 조금 더 격식 있는 톤이 일반적
- 업무 경험, 인사이트, 커리어 관련 주제가 잘 읽힌다
- 첫 1~2줄이 피드 노출의 핵심이므로 임팩트 있게 시작하는 것이 중요

## 분량 및 구조

- 전체 15~25줄 권장
- 한 줄씩 끊어 쓰는 짧은 문장 호흡이 가독성에 유리
- 마지막은 토론을 유도하는 질문이나 의견 제시로 마무리

## 일반적인 글 구조

```
도입 — 직업적 맥락의 질문, 경험, 또는 통찰 한 줄
→ 전개 — 겪은 상황이나 사례 설명
→ 인사이트 — 거기서 얻은 결론이나 시사점
→ 마무리 — 독자가 적용할 수 있는 지점, 또는 질문
```

## 주의사항

- 자기소개형 시작("저는 ~입니다")은 피하는 것이 일반적
- 과도한 실적·수치 과시는 신뢰도를 낮출 수 있음
- 해시태그는 3~5개 정도가 적절
