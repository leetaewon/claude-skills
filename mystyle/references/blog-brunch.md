# 브런치 작법 가이드

## 채널 특성

- 에세이·칼럼 성격이 강한 매체. 정보 전달보다 생각의 흐름이 중심
- 독자는 결론보다 글쓴이의 시선과 과정을 따라가며 읽는 경우가 많다
- 네이버 블로그보다 검색 유입 의존도가 낮고, 글 자체의 매력으로 읽히는 경향

## 분량 및 구조

- 분량: 1,500~3,000자 정도가 일반적
- 소제목은 적게 쓰거나 생략하는 경우가 많음 (흐르는 글 형식)
- 단락 구분은 생각의 전환 지점에서

## 일반적인 글 구조

```
도입 — 글을 쓰게 된 개인적 계기나 상황
→ 전개 — 생각이 흘러가는 과정, 경험
→ 발견 — 그 과정에서 깨닫거나 달라진 것
→ 마무리 — 독자에게 남기는 여운 또는 질문
```

## 제목 작성 시 고려사항

- 정보형 제목("~하는 법")보다 감성적·공감형 제목이 잘 맞는 편
- 호기심을 자극하는 문장형 제목도 자주 쓰임

## 주의사항

- 정보 나열식 구성은 브런치 독자 기대와 맞지 않을 수 있음
- 과도하게 정제된 문장보다 자연스러운 사유의 흐름이 더 잘 읽히는 경우가 많음
