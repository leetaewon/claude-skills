# 페이스북 작법 가이드

## 채널 특성

- 짧고 가벼운 호흡의 글이 잘 읽힌다
- 첫 1~2문장이 "더 보기" 전에 노출되므로 가장 중요한 메시지를 앞에 배치
- 정보 나열보다는 일상적 어조의 단문이 자연스럽다

## 분량 및 구조

- 전체 5~10문장, 10~15줄 이내 권장
- 소제목·번호 매기기 없이 짧은 문단으로 구성
- 마지막 문장은 질문이나 가벼운 제안으로 마무리하면 댓글 유도에 유리

## 일반적인 글 구조

```
도입 — 상황이나 계기 제시 (1~2문장)
→ 전개 — 겪은 일, 생각, 발견 (2~4문장)
→ 마무리 — 정리 또는 독자에게 던지는 질문 (1~2문장)
```

## 주의사항

- 과도한 이모지나 해시태그 사용은 가독성을 떨어뜨릴 수 있음
- "공유해주세요" 같은 직접적 요청은 부담을 줄 수 있어 신중하게
- 너무 긴 문단은 "더 보기" 이후로 밀려 읽히지 않을 가능성이 높음
