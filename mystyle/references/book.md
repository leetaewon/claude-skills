# 책 집필 작법 가이드

## 채널 특성

- 한 번에 끝까지 읽기보다 챕터 단위로 나눠 읽히는 매체
- 블로그·뉴스레터보다 문장의 완결성과 흐름의 일관성이 더 중요
- 독자가 책을 선택한 이유(기대)를 챕터마다 충족시켜야 함

## 분량 및 구조

- 챕터당 분량: 일반적으로 2,000~4,000자
- 챕터 수는 전체 구성에 따라 다르나, 3~5개 단위로 파트를 나누는 경우가 많음
- 소제목은 챕터당 3~5개 정도로 구획

## 일반적인 챕터 구조

```
챕터 오프닝 — 이 챕터에서 다룰 질문이나 상황 제시
→ 본문 — 설명, 사례, 근거를 통한 전개
→ 독자 적용 지점 — 독자가 자기 상황에 대입할 수 있는 부분
→ 챕터 클로징 — 정리 또는 다음 챕터로의 연결
```

## 주의사항

- "이 책에서는 ~를 다룹니다" 같은 교재형 선언은 도입부에서 반복되면 지루하게 느껴질 수 있음
- 결론을 먼저 말하고 근거를 나중에 설명하는 구조가 책에서는 다소 단조롭게 읽힐 수 있음
- 실제 경험과 가상의 예시는 구분해서 명시하는 것이 신뢰도에 도움이 됨
