# 뉴스레터 작법 가이드

## 채널 특성

- 1:1로 받는 느낌을 주는 매체. 공지문보다는 편지 같은 어조가 자연스럽다
- 모바일에서 읽는 경우가 많아 짧은 문단이 유리하다
- 한 호에 너무 많은 내용을 담으면 핵심이 흐려진다

## 분량 및 구조

- 한 호에 하나의 핵심 주제만 다루는 것을 권장
- 문단은 2~3줄 이내로 짧게 끊기
- 인사말로 시작하기보다 바로 본론으로 들어가는 편이 몰입도가 높다
- 마지막에 링크나 행동 유도(CTA)는 하나로 제한

## 일반적인 글 구조

```
오프닝 — 최근 있었던 일이나 화두 (1~2문장)
→ 본문 — 발견한 것, 전하고 싶은 이야기
→ 클로징 — 정리 또는 독자에게 묻는 질문 하나
```

## 주의사항

- 여러 섹션으로 나눈 "소식 모음" 형태는 핵심이 분산되기 쉽다
- 소제목·번호 매기기보다 흐르는 문장으로 구성하는 편이 편지 느낌에 가깝다
- 발신 빈도와 분량의 균형을 고려 (너무 길면 완독률이 떨어짐)
