# 카드뉴스 작법 가이드

## 채널 특성

- 한 장(카드)에 하나의 메시지만 담는 것이 원칙
- 텍스트보다 시각적 임팩트가 우선되는 매체
- 스크롤하며 빠르게 훑어보는 소비 패턴을 고려해야 함

## 유형이 불분명하면

작성 전에 "정보 전달형(외부 이슈나 트렌드 해설)"인지 "하이라이트 요약형(자기 콘텐츠를 압축해서 보여주는 것)"인지 먼저 확인한다.

## 분량 및 구조

- 카드 수: 8~10장 권장, 정보량에 따라 조정
- 카드별 텍스트: 헤드라인 + 본문 1~3줄 이내

## 1) 정보 전달형

```
표지 — 이슈를 한 줄로 제시
→ 배경 설명
→ 핵심 정보 (1~3장)
→ 시사점 또는 다른 시각
→ 정리 (출처가 있다면 명시)
```

## 2) 하이라이트 요약형

```
표지 — 핵심 메시지를 한 줄로
→ 문제 제기
→ 전환점
→ 핵심 정리 (1~3장)
→ 다음 행동 안내 (자세한 내용을 볼 수 있는 곳 안내)
```

## 주의사항

- 한 카드에 너무 많은 정보를 담지 않기
- 같은 문장 구조를 반복하면 단조롭게 느껴질 수 있음
- 출처나 통계를 인용할 때는 과장 없이 정확하게
