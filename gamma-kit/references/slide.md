# 강의 슬라이드 제작 가이드

## Gamma 설정

| 항목 | 값 |
|---|---|
| 제작 방식 | `generate` |
| design_type | `presentation` |
| themeId | (없음 — Gamma 기본 테마 적용. 본인 테마 ID 있으면 여기에 추가) |
| 언어 | `ko` |

## 슬라이드 구성 원칙

- 장수: 10~20장 권장 (강의 길이에 따라 조정)
- 1장당 1개념. 텍스트 과적 금지
- 오프닝: 오늘 배울 것을 "상황"으로 제시 (제목 나열 금지)
- 클로징: 핵심 1줄 정리 + 다음 단계 or 질문

## 슬라이드 흐름 구조

```
표지 — 강의 제목 + 발표자명
→ 오늘의 상황 제시 (왜 이걸 배우는가)
→ 본론 슬라이드 (개념 → 실습 or 예시 반복)
→ 핵심 정리 1장
→ 다음 단계 or Q&A 유도
```

## 텍스트 톤

- 슬라이드 헤드라인: 짧고 단정하게 (반말 ~다체 or 명사형)
- 본문 설명: 해요체 대화체 허용 (강의 맥락)
- "~하는 법", "~가이드" 식 제목 금지

## Gamma 생성 시 inputText 작성 요령

- 슬라이드별 제목 + 핵심 내용을 간략히 정리해서 전달
- 이미지 소스: `aiGenerated` 기본 사용
- textMode: `generate` (아웃라인 기반으로 확장)

## 주의사항

- 슬라이드에 개인 연락처·가격 정보 직접 노출 금지
- 웨비나용 슬라이드는 화면 공유 기준으로 16:9 비율 유지
