# 카드뉴스 제작 가이드

## mystyle 연동

카드뉴스 제작 시 mystyle의 `references/card-news.md`를 함께 로드하여 문체 가이드를 적용한다.
유형(뉴스해설형 / 요약형)도 mystyle 파일의 기준을 따른다.

## Gamma 설정

| 항목 | 값 |
|---|---|
| 제작 방식 | `generate` |
| design_type | `instagram_post` (1080×1350px, 세로형) |
| themeId | (없음 — Gamma 기본 테마 적용. 본인 테마 ID 있으면 여기에 추가) |
| templateId | (없음 — 전용 템플릿 있으면 여기에 추가하고 제작 방식을 `generate_from_template`으로 변경) |
| 언어 | `ko` |

## 카드 구성 원칙

- 장수: 8~10장 권장, 정보량에 따라 유연하게
- 1장당 1메시지. 설명보다 임팩트 우선
- 표지(1장) + 본문(6~8장) + CTA(1장) 기본 구조

## 유형별 구조

### 뉴스해설형
```
표지 — 이슈 한 줄 (반말 단정형)
→ 배경 (1장)
→ 핵심 정보 (1~3장, 정보박스)
→ 비판적 시각 / 반론 (1장)
→ 정리 + 출처 (1장)
```

### 요약형
```
표지 — 선언적 한 줄 (반말 단정형)
→ 문제 제시 (1장)
→ 전환 (1장)
→ 핵심 정리 (1~3장)
→ CTA — 블로그 or 채널 링크 (1장)
```

## Gamma 생성 시 inputText 작성 요령

- 카드별 헤드라인 + 본문 1~3줄을 카드 단위로 구분해서 전달
- cardSplit: `inputTextBreaks` 사용 (카드 경계를 직접 지정)
- 이미지 소스: 요약형은 `pictographic` or `aiGenerated`, 뉴스해설형은 `aiGenerated`
- textMode: `preserve` (카드 단위 내용 그대로 유지)

## 주의사항

- 해요체("~입니다", "~하세요") 카드 헤드라인에 사용 금지
- CTA 문구("~에서 확인하세요")는 예외적으로 해요체 허용
- 출처·통계 인용 시 과장 표현 금지
