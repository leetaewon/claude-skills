# claude-skills

이태원쌤이 직접 사용하는 Claude 스킬 모음.

콘텐츠 제작, 지식관리, 1인기업 워크플로우를 위해 만든 스킬들을 정리해서 공개합니다.

---

## 스킬 목록

| 스킬 | 설명 |
|---|---|
| [repozer](./repozer) | GitHub 저장소의 `CLAUDE.md`, `index.md`를 버전 비교 기반으로 동기화 |
| [idea](./idea) | 주제 하나로 7가지 각도의 콘텐츠 아이디어 생성 |
| [mystyle](./mystyle) | 채널별(뉴스레터, 페이스북, 링크드인, 네이버 블로그, 브런치, 카드뉴스, 유튜브, 책) 작법 가이드 |
| [style-creator](./style-creator) | 인터뷰 기반으로 나만의 문체 스킬(mystyle) 생성 |
| [gamma-kit](./gamma-kit) | Gamma MCP로 슬라이드, 카드뉴스, 전자책, 리드마그넷 제작 |

---

## 설치 방법

각 스킬 폴더의 README.md에서 `.skill` 파일을 다운로드한 뒤, Claude 프로젝트 → 사용자 지정 → 스킬 만들기 → 스킬 업로드 화면에서 업로드하면 됩니다.

---

## mystyle ↔ style-creator 관계

`mystyle`은 채널별 일반 작법만 담은 범용 스킬입니다. `style-creator`로 인터뷰를 진행하면 나만의 문체가 반영된 맞춤형 `mystyle`을 새로 만들어 교체할 수 있습니다.
